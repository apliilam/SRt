StockRelease tool (SRt) - конструктор для заполнения универсальной формы Релизов (Model Release, Property Release).

Лицензия: 
   GPLv3

Требования:
 - разрешение экрана 1600х900;
 - Операционная система — Windows7 или выше.

Доступные формы:
 - Model Release (английская, русская);
 - Property Release (английская, русская).

Возможности:
 - создание Model Release и Property Release;
 - поддержка StockSubmitter; 
 - сохранение персональных данных;
 - создание одного Property Release для 5 файлов, сделанных по одному референсу;
 - автоматическое создание Property Release для каждого файла;
 - добавление нескольких изображений(референсов) в одном Release; 
 - возможность редактирования изображений (поворот, масштабирование, обрезка, расположение);
 - автоматическое заполнение даты из файлов и референсов (для Property Release).
